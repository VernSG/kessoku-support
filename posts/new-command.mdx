## 🎮 Command `quiz-anime`

Command `quiz-anime` adalah command interaktif yang memberikan kuis tentang anime kepada pengguna dalam format pilihan ganda. Pengguna akan mendapatkan satu pertanyaan acak seputar anime, dan mereka bisa menjawab dengan mengklik salah satu tombol jawaban yang disediakan.

> Jika jawaban benar, pengguna akan mendapatkan **+1 poin** yang otomatis ditambahkan ke total skor mereka. Poin ini disimpan di database dan digunakan untuk mengukur peringkat mereka di leaderboard.

### Cara Menggunakan `quiz-anime`

Cukup ketik `/quiz-anime` di chat server Discord, dan bot akan menampilkan pertanyaan pilihan ganda lengkap dengan tombol jawaban. Berikut adalah fitur utama dari command ini:

- **Pertanyaan Acak**: Setiap kali pengguna menjalankan command, bot akan memberikan pertanyaan acak dari database kuis anime.
- **Interaksi Tombol**: Jawaban diberikan melalui tombol pilihan, sehingga pengguna bisa langsung memilih tanpa perlu mengetik ulang.
- **Sistem Poin Otomatis**: Pengguna yang menjawab dengan benar akan menerima poin tambahan yang disimpan di sistem leaderboard.

## 🏆 Command `leaderboard`

Command `leaderboard` memungkinkan pengguna untuk melihat peringkat teratas berdasarkan poin yang diperoleh dari kuis anime. Command ini mempromosikan kompetisi yang sehat di antara para pengguna dan mendorong mereka untuk terus menjawab kuis dengan benar.

### Cara Menggunakan `leaderboard`

Ketik `/leaderboard` di chat server Discord, dan bot akan menampilkan daftar pengguna dengan poin tertinggi dalam bentuk embed. Berikut adalah fitur dari command ini:

- **Peringkat 10 Teratas**: Menampilkan daftar pengguna dengan poin tertinggi.
- **Sistem Poin Terintegrasi**: Poin yang diperoleh dari `quiz-anime` akan otomatis dihitung dalam leaderboard.
